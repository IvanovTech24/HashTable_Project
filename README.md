**HashTable Project** 📚
This project is a custom implementation of a Hash Table in Python. 
It handles hash collisions using **Linear Probing** and includes dynamic resizing.

**Features** 🚀
🔑 **Hashing & Linear Probing**: Resolves collisions by checking the next available slot.

🗑️ **Deletion Support**: Uses a special Deleted object to maintain the probing chain.

🛠️ **Dynamic Resizing**: Automatically increases capacity when the table fills up.
